# E2EE-Loader-
Auto Message Sender For Messenger

Auto Message Sender For Messenger is a lightweight browser extension designed to automate message sending on Messenger Web. It helps users quickly send predefined messages, repeat texts, and manage simple messaging tasks directly from the browser.

Features

- Auto send custom messages
- Fast Messenger Web integration
- Lightweight and easy to use
- Popup-based control panel
- Quick message repeat system
- Live key and config support via GitHub
- Works with supported Chromium browsers

How It Works

The extension runs on Messenger Web and allows users to trigger automated text sending using predefined settings. Configuration files and updates can be managed remotely through GitHub hosting.

Supported Browsers

- Kiwi Browser
- Chromium-based browsers
- Other compatible extension-supported browsers

Notes

- Requires Messenger Web access
- Internet connection required for live config updates
- Some features may depend on browser extension permissions

Version

Current Version: 1.6
